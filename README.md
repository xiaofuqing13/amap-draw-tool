# 高德地图绘制与坐标提取

这是一个本地可运行的单页工具：
- 在高德地图上绘制多边形 / 矩形 / 圆 / 折线（折线会自动闭合并带填充色）
- 自动提取并展示所有图形的经纬度
- 每次变更自动保存到本地文件，下次打开自动恢复

## 目录结构

- `index.html`：前端页面（地图绘制与坐标提取）
- `server.py`：本地服务（提供读写接口并持久化到文件）
- `overlays.json`：自动保存的图形数据

## 本地运行

在项目目录执行：

```bash
cd /Users/xiaofuqing/Desktop/画图
python3 server.py
```

浏览器打开：

- http://127.0.0.1:4173/

建议首次打开后强制刷新一次：
- macOS：`Cmd + Shift + R`

## 数据持久化说明

- 前端会在以下时机自动保存到 `overlays.json`：
  - 新绘制完成
  - 删除图形
  - 清空图形
- 页面加载时会从 `overlays.json` 自动恢复并显示。

## 注意事项

- 请使用 `http://127.0.0.1:4173/` 打开页面，不要直接双击 `index.html`（`file://`）。
- 当前已在 `index.html` 中写入高德 JS API Key 与安全密钥，仅建议在内网或本地工具使用。
